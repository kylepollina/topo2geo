
# topo2json

CLI for converting TopoJSON files to GeoJSON

### Installation

```
pip install topo2geo
```

Depends on [shapely](https://pypi.org/project/Shapely/)

### Usage

```
[~]$ topo2geo input_topo.json output_geo.json
```
